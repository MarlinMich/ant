1. Określanie właściwości obiektu:
- Poprzez bezpośredni dostęp do pola:
```javascript
let samochod = { marka: "Toyota", model: "Camry" };
let markaSamochodu = samochod.marka; // dostęp do pola "marka"
```
- Za pomocą metody get w klasie (w przypadku języków wspierających getter):
```javascript
class Samochod {
constructor(marka) {
this._marka = marka;
}
get marka() {
return this._marka;
}
}
let mojSamochod = new Samochod("Toyota");
let marka = mojSamochod.marka; // dostęp do pola "marka" przez metodę get
```

2. Odwoływanie się do pól (właściwości) i metod:
- Odwołanie do pola:
```javascript
let wartośćPola = obiekt.pole;
```
- Wywołanie metody:
```javascript
obiekt.metoda();
```

3. Cechy konstruktora:
- Inicjalizuje nowo tworzony obiekt.
- Często używany do ustawiania początkowego stanu obiektu.
- Ma ten samą nazwę co klasa, do której należy.

4. Konstruowanie obiektu:
```javascript
class Przyklad {
constructor(pole1, pole2, pole3) {
this.pole1 = pole1;
this.pole2 = pole2;
this.pole3 = pole3;
}
metoda() {
// kod metody
}
}
let nowyObiekt = new Przyklad("wartosc1", "wartosc2", "wartosc3");
```

5. Znaczenie słowa "this":**
- "this" odnosi się do bieżącego obiektu.
- Oddziela pola od parametrów i zmiennych lokalnych.
```javascript
class Przyklad {
constructor(pole) {
this.pole = pole; // "this" oznacza, że przypisujemy wartość do pola obiektu
}
}
```

6. Konstruktor w JavaScript (JS):**
```javascript
function Samochod(marka, model) {
this.marka = marka;
this.model = model;
}
let mojSamochod = new Samochod("Toyota", "Camry");
```

7. Znaczenie słowa kluczowego "new":**
- "new" tworzy nową instancję obiektu.
```javascript
let nowyObiekt = new Klasa();
```

8. Znaczenie słowa kluczowego "prototype":
- "prototype" jest obiektem, który zawiera właściwości i metody, które będą współdzielone przez wszystkie instancje danej klasy.
```javascript
function Klasa() {}
Klasa.prototype.metoda = function() {
// kod metody
};
let obiekt1 = new Klasa();
let obiekt2 = new Klasa();
obiekt1.metoda(); // Wywołanie metody dla obiekt1
```
